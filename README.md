# API Imprimir - Backend Flask

API REST desarrollada en **Python (Flask)** para sistema de gestión de órdenes, empleados y procesos de impresión.

## Tecnologías
- Python
- Flask
- SQLite
- API REST

## Funcionalidades
- Autenticación de usuarios
- Gestión de empleados
- Gestión de órdenes
- Procesamiento de datos
- Automatización de procesos
- Integración con frontend web
- Servicios de impresión

## Estado del proyecto
Proyecto en desarrollo activo, actualmente **operativo en dos locales comerciales reales**, en proceso de mejora continua e implementación de buenas prácticas.

## Proyecto personal
Sistema desarrollado como proyecto personal productivo, aplicado en entornos reales de operación comercial.
